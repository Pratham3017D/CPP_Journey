# Mission 2030

> Become the kind of software engineer who earns the freedom to choose
> where to work, where to live, and the life to build.

## Vision

Graduate in 2029 ready for: - Elite Product Companies - Global
Capability Centers (GCCs) - AI Engineering (optional) - International
Opportunities - High-paying Software Engineering Jobs

## Core Principles

1.  Engineering First
2.  Strong Fundamentals (C++, DSA, OS, DBMS, CN, OOP, System Design)
3.  Build Real Things (Projects \> Certificates)
4.  Build in Public (LinkedIn, X, GitHub, Portfolio)
5.  AI is Your Teammate
6.  Learn Forever

## Year 1 - Foundation

-   C++, Python basics, JavaScript
-   Arrays, Strings, Functions, Recursion, Pointers, STL
-   HTML, CSS, React
-   Git, GitHub, VS Code, Linux Basics
-   Portfolio Website
-   Mission 2030 Dashboard
-   3-5 Projects
-   Weekly LinkedIn Posts
-   High CGPA, Coding Club, Hackathons

## Year 2 - Builder

-   Node.js, Express
-   MongoDB, PostgreSQL, Redis
-   Docker, AWS Basics, CI/CD
-   First Open Source Contribution
-   6-10 Projects
-   AI Orchestration Basics
-   First Internship

## Year 3 - Elite Engineer

-   Advanced DSA
-   System Design
-   Distributed Systems
-   Kubernetes
-   LLMs, RAG, Agents
-   Elite Internship
-   Leadership

## Year 4 - Freedom

-   Interview Preparation
-   Resume & Portfolio
-   PPO / Global Companies / GCCs
-   Product, Backend or AI Engineering

## Weekly Routine

-   Program Daily
-   Build Weekly
-   1 LinkedIn Post / Week
-   2-3 X Posts / Week
-   Weekly GitHub Commits
-   Read Documentation
